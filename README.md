# RVARC Field Day 2026 — Public Information Website

Website for the Rappahannock Valley Amateur Radio Club's Field Day 2026 Public Information Table.

**Live site:** https://[your-username].github.io/rvarc-field-day/

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Home — event details and site overview |
| `table-plan.html` | Coordinator guide for the Public Information Table |
| `about-rvarc.html` | Club history, mission, repeater, meetings |
| `get-licensed.html` | License classes, study resources, finding an exam |
| `sign-up.html` | Visitor sign-up form (mailto-based) |
| `downloads.html` | All printable documents + ARRL brochure links |

## Downloads folder

`downloads/` contains printable PDF and editable Word (.docx) versions of all four RVARC-produced documents.

## Enabling GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Under **Source**, select **Deploy from a branch**
4. Choose `main` branch, `/ (root)` folder
5. Click **Save** — your site will be live in ~60 seconds at `https://[username].github.io/[repo-name]/`

## Updating for future events

- Edit the HTML files directly to update dates, contacts, or content
- Replace files in `downloads/` with updated PDFs/docx as needed
- The sign-up form on `sign-up.html` sends to `info@rvarc.club` via mailto — update that address if needed
